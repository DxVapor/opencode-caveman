# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a Vulnerability

Do **not** open a public GitHub issue for security vulnerabilities.

Report privately via [GitHub's private vulnerability reporting](https://github.com/DxVapor/opencode-caveman/security/advisories/new) or email the maintainer directly.

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You'll receive a response within 72 hours. If confirmed, a fix will be prioritized and a new patch version released.
